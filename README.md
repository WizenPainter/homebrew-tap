# Homebrew tap for Shuffle

[Shuffle](https://shuffleapp.co) is a fast, open-source, GPU-rendered file
manager for macOS - a Finder alternative built in Rust with GPUI.

## Install

```sh
brew install wizenpainter/tap/shuffle
```

Requires macOS 12 (Monterey) or later. The app is signed and notarized.

## Update

```sh
brew upgrade shuffle
```

Source code: https://github.com/WizenPainter/shuffle
